# THE YOKAI ⚔️👹

夜の東京を舞台にした3Dアクションゲーム。妖怪を倒して魂を集め、キャラクターを強化しよう！

🎮 **プレイ:** https://the-yokai-production.up.railway.app

## 概要

- **ジャンル:** 3Dアクション
- **技術:** SvelteKit + Threlte (Three.js) + TypeScript
- **デプロイ:** Railway

## 機能

### ✅ 実装済み
- 三人称視点カメラ（マウスで回転）
- カメラ基準の移動（WASD）
- 左クリックで攻撃
- **Valorant風スキルシステム（Q/E/R）**
- 4キャラクター（刀・銃・術・格闘）
- キャラ別スキルエフェクト
- クールダウン＆Ultチャージ
- 敵（小鬼）との戦闘
- 魂の収集システム
- セーブ機能（localStorage）
- 東京の夜ステージ（ネオン・自販機）

### 🔜 未実装（予定）
- 攻撃モーション改善
- スキルのダメージ処理
- 敵の攻撃パターン
- キャラクターデザイン改善
- ボス敵
- BGM・効果音

## 操作方法

| キー | 操作 |
|------|------|
| WASD | 移動（カメラ基準） |
| マウス | カメラ回転（クリックでロック） |
| 左クリック | 攻撃 |
| Q | スキル1（クールダウン8秒） |
| E | スキル2（クールダウン12秒） |
| R | Ultimate（敵撃破でチャージ） |
| 1-4 | キャラ切り替え |
| K | スキル詳細 |
| C | キャラ選択 |

## キャラクター＆スキル

| キャラ | 名前 | Q | E | R |
|--------|------|---|---|---|
| 🗡️ | 刀使い・零 | 居合斬り | 剣気 | 一閃 |
| 🔫 | 銃使い・凛 | 連射 | 閃光弾 | 必中 |
| ✨ | 術師・紫 | 火球 | 結界 | 隕石 |
| 👊 | 格闘家・剛 | 瞬歩 | 練気 | 百裂拳 |

- 🗡️ 初期キャラ
- 🔫 50魂で解放
- ✨ 100魂で解放
- 👊 75魂で解放

## 開発

```bash
# 依存関係インストール
npm install

# 開発サーバー
npm run dev

# ビルド
npm run build

# デプロイ
railway up --service the-yokai
```

## ファイル構成

```
src/
├── lib/
│   ├── components/
│   │   ├── Game.svelte          # ゲームコンテナ
│   │   ├── Scene.svelte         # 3Dシーン
│   │   ├── ThirdPersonCamera.svelte  # カメラ
│   │   ├── Stage.svelte         # ステージ
│   │   ├── Player.svelte        # プレイヤー
│   │   ├── Enemy.svelte         # 敵
│   │   └── GameUI.svelte        # UI（スキルバー）
│   └── game/
│       ├── gameState.ts         # 状態管理
│       └── skillTrees.ts        # スキル定義
└── routes/
    └── +page.svelte             # メインページ
```
